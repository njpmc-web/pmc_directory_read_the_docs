# Edits and Verification

## Goal

Create a phone-change request and immediately verify it with a code delivered to the new phone.

## Before you begin

- Be signed in on [Family information](family.md).
- Be able to receive SMS messages at the new phone.

## Steps

1. In the member’s **휴대폰** (phone) row, choose **수정** (edit).
2. Enter the country code and new phone, then choose **요청** (request). **취소** closes the editor and keeps the current value.

3. In **인증번호를 받을까요?** (receive a verification code?), confirm that the displayed new phone is correct and choose **지금 인증** (verify now).

4. In **휴대폰 인증** (phone verification), enter the **6자리 인증번호** delivered to the new phone and choose **인증하기** (verify).

5. Confirm **휴대폰 인증이 완료되었습니다.** (phone verification is complete) at the top. The phone row shows `Requested:`, a `Verified` badge, and the requested phone.
6. Under **요청 내역** (request history), confirm the same request’s `Request #number`, `담당: 교적 관리자` (responsible: directory administrator), before/after phones, and `Verified` badge.

## Successful result

The phone-change request is at **접수** (received), and phone verification shows `Verified`. `Verified` confirms the new phone; it does not mean that staff have finished applying the directory change.
