# Edits and Verification

## Goal

Request a new English name, phone, email, or address and complete required contact verification.

## Before you begin

- Be signed in on the family page.
- Be able to receive a code at the new phone/email.

## Steps

1. Choose **정보 수정** on a member card and open the field.
2. Enter the new value. For an address, select the exact search result.
3. Choose **수정 요청 보내기** and confirm the before/after values.
4. For phone/email, enter the code sent to the new contact. If deferred, verify later from request details under **요청 내역**.

## Successful result

A request number appears with **접수** status. Phone/email requests also show verification completion.

## Common problems

- **Same/invalid value:** enter a valid value different from the current one.
- **No address result:** begin with street number/name, then confirm city/state.
- **No later-verification button:** the contact may already be verified or the request may no longer be eligible.
