# Photo Requests

## Goal

Select and crop an individual or family photo, then submit a replacement request.

## Before you begin

- Prepare a clear JPG, PNG, or HEIC image.
- Obtain consent from family members shown.

## Steps

1. Choose **사진 변경 요청** on the individual or family photo card.
2. Select an image, then zoom/move it to set the crop.
3. Choose **이 사진 사용**, review the preview, and submit.
4. For a wrong photo, cancel the pending request before requesting again.

## Successful result

Request history shows **개인 사진** or **가족 사진** with a preview.

## Common problems

- **Unreadable file:** convert to JPG/PNG or use a smaller image.
- **Face cropped:** leave space around faces before accepting the crop.
- **Cannot request again:** cancel the existing pending request for that target.
