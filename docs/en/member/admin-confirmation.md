# Admin Confirmation Request

## Goal

When sign-in cannot find your registered phone, verify the phone and submit identity details for administrator review.

## Before you begin

- You need your own phone that can receive SMS messages.
- Name and address are required. Prepare spouse name, directory number, and birth date if known.

## Steps

1. If member sign-in shows **등록된 교인 휴대폰 번호를 찾을 수 없습니다.** (registered member phone not found), choose **관리자 확인 요청** (admin confirmation request).

2. Under **휴대폰 본인 인증** (phone identity verification), confirm or enter the phone and choose **인증번호 받기**.

3. Enter the SMS code in **인증번호**, then choose **번호 인증** (verify phone).

4. Under **본인 정보 확인** (confirm identity details), confirm **인증한 전화번호** (verified phone).
5. Enter **이름** (name) and **주소** (address). If address search is available, select the exact suggestion to fix the address; choose **변경** to replace it. If search is unavailable, enter the full address and ZIP manually.
6. If known, enter **배우자 이름 (선택)** (spouse name), **교적번호 (선택)** (directory number), and **생년월일 (선택)** (birth date). Add any explanation in **관리자에게 보낼 일반 메모 (선택)** (optional note to the administrator).

7. If **일치 후보** (matching candidates) appear, select the person you believe is you. If none is correct, select **후보를 선택하지 않고 관리자 검토 요청** (request administrator review without selecting a candidate).

8. Choose **관리자 검토 요청 제출** (submit administrator review request).
9. Confirm **요청 #번호이 접수되었습니다. 처리 결과는 인증한 휴대폰 번호로 안내합니다.** and save the request number. This says the request was received and the result will be sent to the verified phone.

## Successful result

The app issues an administrator confirmation request number and identifies the phone that will receive the result.

## Next step

Wait for the result SMS at the verified phone. After an approval notice, return to [Member sign-in](login.md) and try again. If progress stops, use [Get support](../support.md) with the request number.
