# Get Support

Send the support team:

1. Member or admin screen
2. Time and browser/device
3. Request number, if any
4. Exact on-screen error
5. Troubleshooting already attempted

Copyable template:

```text
Screen/role:
Time:
Request number:
Error text:
Actions tried:
```

!!! warning "Do not send"
    Never attach an OTP, password, full phone/address, family photo, or admin sign-in link.
