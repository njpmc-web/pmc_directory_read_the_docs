# PMC Directory User Manual

Learn how members use **PMC 교인 정보 확인** (PMC Directory) to review family details and submit changes, and how staff review those requests.

## Where should I start?

- New members: [Quick start](quick-start.md) and [Member sign-in](member/login.md).
- Profile changes: [Edits and verification](member/edit.md); photo changes: [Photo requests](member/photos.md).
- Staff: check [Access by role](access.md), then open the relevant admin procedure.
- If you are stuck, see [Troubleshooting](troubleshooting.md) or [Get support](support.md).

!!! privacy "All examples are synthetic"
    Names, contact details, photos, and request numbers shown in this manual do not identify real members.
