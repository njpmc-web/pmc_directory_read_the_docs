# Feature Overview

| Area | What you can do |
|---|---|
| Member sign-in | Registered phone/email OTP; request help for an unregistered number |
| Family information | Review family members, contacts, address, individual and family photos |
| Change requests | Request changes to English name, phone, email, address, and photos |
| Request history | Review status/details, cancel eligible requests, delete canceled history, verify contacts later |
| Administration | Search members, link unregistered numbers, process requests, manage logs, photo settings, and roles |

The normal status path is **접수** (received) → **확인 중** (under review) → **입력 완료** (entered) → **반영 확인** (verified). An unworkable request is **반려** (rejected); a withdrawn request is **취소** (canceled).
