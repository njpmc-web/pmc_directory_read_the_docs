# Glossary

| Term | Meaning |
|---|---|
| OTP | A six-digit one-time verification code |
| 교적 | Church-managed member/family directory record |
| 접수 | Submitted and waiting for review |
| 확인 중 | Under staff review |
| 입력 완료 | Staff entered the change |
| 반영 확인 | Change verified in the directory/app |
| 반려 | Closed without change, with a reason |
| Lookup | Admin function linking unregistered-number requests to candidates |
| Deferred verification | Verifying a new phone/email later from request details |
