# Process Requests

## Goal

Review all profile requests and accurately update status and staff notes.

## Before you begin

- Super Admin is required.
- You must be able to verify the actual directory update.

## Steps

1. In **Requests**, filter by status/field and select a request.
2. Compare requester, target member, before/after value, photo, and verification state.
3. Set **확인 중** while reviewing and **입력 완료** after entering the change.
4. Set **반영 확인** after verification, or **반려** with a reason if it cannot be processed.

## Successful result

Member request details show the newest status, responsible role, and processing note.

## Common problems

- **Unverified contact:** check verification before processing phone/email.
- **Photo comparison unavailable:** review only in approved systems; do not retain it on a personal device.
