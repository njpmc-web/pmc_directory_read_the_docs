# Unregistered-number Review

## Goal

Link an unregistered-number sign-in request to a directory candidate, then approve or reject it.

## Before you begin

- Super Admin or Lookup Admin is required.
- Never decide from user-entered information alone.

## Steps

1. In **Lookup**, open a pending request and review name, phone, and progress.
2. Compare candidate location, phone ending, and family relationship.
3. Select a confirmed candidate and choose **연결하고 승인**.
4. If no candidate can be confirmed, record a reason and choose **요청 반려**.

## Successful result

An approved request links to the correct record; a rejected request retains its reason.

## Common problems

- **Several candidates:** do not guess; obtain further confirmation.
- **Already processed:** refresh to see whether another admin completed it.
