# Troubleshooting

| Symptom | Check | Next action |
|---|---|---|
| OTP not received | Registered contact, spam, newest request | Try the other channel |
| Unregistered number | Country code and digits | Request admin review |
| Wrong family information | Close screen and sign out | Report as a privacy incident |
| Photo upload failed | Format, size, network | Retry with smaller JPG/PNG |
| Request submission failed | Input, network, and whether a request appeared in history | Check history before retrying to avoid duplicates |
| Need to cancel or replace a request | Current card status and available controls | Use **요청 취소** while it is at **접수**, then submit again |
| Request rejected | Card status and note | Keep the request number and note; contact support if needed |
| Admin menu missing | Current account and role | Ask Super Admin to verify access |
| Request stalled | Request number and last change | [Get support](support.md) |

Before refreshing, safely note any non-sensitive input you need. Never attach verification codes or family photos to a support request.
