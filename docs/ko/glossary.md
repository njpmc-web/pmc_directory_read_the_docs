# 용어집

| 용어 | 뜻 |
|---|---|
| OTP | 한 번만 쓰는 6자리 인증번호 |
| 교적 | 교회가 관리하는 회원·가족 정보 기록 |
| 접수 | 요청이 제출되어 검토를 기다림 |
| 확인 중 | 관리자가 요청을 검토 중 |
| 입력 완료 | 관리자가 변경을 입력함 |
| 반영 확인 | 실제 화면/교적에 변경이 반영됨 |
| 반려 | 요청을 처리할 수 없어 사유와 함께 종료 |
| Lookup | 미등록 전화번호 요청을 교적 후보와 연결하는 관리자 기능 |
| 사후 인증 | 요청 제출 뒤 요청 상세에서 새 전화·이메일을 인증하는 절차 |
