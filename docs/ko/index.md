# PMC Directory 사용자 매뉴얼

PMC Directory(화면의 **PMC 교인 정보 확인**)에서 가족 정보를 확인하고 수정 요청을 보내는 방법과, 직원이 요청을 검토하는 방법을 안내합니다.

## 어디에서 시작할까요?

- 처음 이용하는 회원은 [빠른 시작](quick-start.md)과 [회원 로그인](member/login.md)을 보세요.
- 가족 정보 변경은 [정보 수정 및 인증](member/edit.md), 사진 변경은 [사진 요청](member/photos.md)을 보세요.
- 직원은 [역할별 접근 권한](access.md)을 확인한 뒤 해당 관리자 절차로 이동하세요.
- 막혔다면 [문제 해결](troubleshooting.md) 또는 [지원 요청](support.md)을 이용하세요.

!!! privacy "실제 교인 정보가 아닌 예시입니다"
    이 매뉴얼의 이름, 연락처, 사진, 요청 번호는 모두 합성 예시입니다.
