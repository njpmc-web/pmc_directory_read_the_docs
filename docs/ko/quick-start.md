# 빠른 시작

## 회원

1. **SMS 로그인** 또는 **이메일로 로그인**을 선택합니다.
2. 등록된 연락처로 받은 6자리 인증번호를 입력합니다.
3. **우리 가족 정보**에서 가족과 사진을 확인합니다.
4. 바꿀 항목의 **정보 수정**을 선택해 요청합니다.
5. **요청 내역**에서 진행 상태를 확인합니다.

## 직원

1. `/admin/login`에서 승인된 Google 계정으로 로그인합니다.
2. 역할에 허용된 메뉴를 엽니다.
3. 요청 상세와 교적 정보를 대조한 후 처리합니다.
