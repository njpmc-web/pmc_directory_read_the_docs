# 정보 수정 및 인증

## 목적

영문 이름, 휴대폰, 이메일 또는 주소 변경을 요청하고 필요한 연락처 인증을 마칩니다.

## 사전 조건

- 가족 화면에 로그인되어 있어야 합니다.
- 새 휴대폰이나 이메일로 인증번호를 받을 수 있어야 합니다.

## 작업 단계

1. 구성원 카드에서 **정보 수정**을 선택하고 수정 항목을 엽니다.
2. 새 값을 입력합니다. 주소는 검색 결과에서 정확한 주소를 선택합니다.
3. **수정 요청 보내기**를 선택하고 변경 전·후 값을 확인합니다.
4. 휴대폰 또는 이메일 요청은 안내에 따라 새 연락처로 받은 인증번호를 입력합니다. 즉시 못했다면 **요청 내역** 상세에서 나중에 인증할 수 있습니다.

## 성공 결과

새 요청 번호가 만들어지고 **접수** 상태로 표시됩니다. 전화·이메일은 인증 완료 표시도 확인합니다.

## 흔한 오류와 해결

- **같은 값/잘못된 형식**: 현재 값과 다른 유효한 값을 입력합니다.
- **주소 검색 결과 없음**: 번지와 도로명을 먼저 입력하고 도시·주를 확인합니다.
- **사후 인증 버튼 없음**: 이미 인증되었거나 처리할 수 없는 상태인지 요청 상세를 확인합니다.
