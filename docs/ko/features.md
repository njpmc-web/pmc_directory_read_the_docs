# 기능 개요

| 영역 | 할 수 있는 일 |
|---|---|
| 회원 로그인 | 등록된 휴대폰 또는 이메일 OTP 로그인, 미등록 번호 확인 요청 |
| 가족 정보 | 가족 구성원, 연락처, 주소, 개인·가족 사진 확인 |
| 수정 요청 | 영문 이름, 휴대폰, 이메일, 주소 및 사진 변경 요청 |
| 요청 내역 | 상태·상세 확인, 가능한 요청 취소, 취소 기록 삭제, 사후 인증 |
| 관리자 | 교인 검색, 미등록 번호 연결, 요청 처리, 로그인 기록·사진 설정·권한 관리 |

상태는 보통 **접수 → 확인 중 → 입력 완료 → 반영 확인** 순서입니다. 처리할 수 없으면 **반려**, 회원이 철회하면 **취소**로 표시됩니다.
