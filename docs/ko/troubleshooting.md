# 문제 해결

| 증상 | 확인할 내용 | 다음 조치 |
|---|---|---|
| OTP가 오지 않음 | 등록 연락처, 스팸함, 최신 요청 여부 | 다른 로그인 채널 시도 |
| 등록되지 않은 번호 | 국가번호와 숫자 오류 | 관리자 확인 요청 |
| 가족 정보가 이상함 | 즉시 화면을 닫고 로그아웃 | 개인정보로 취급해 지원팀 신고 |
| 사진 업로드 실패 | 형식, 파일 크기, 네트워크 | JPG/PNG 소형 파일로 재시도 |
| 요청 제출 실패 | 입력값과 네트워크, 요청이 목록에 생성되었는지 | 중복 제출하지 말고 목록 확인 후 재시도 |
| 요청 취소/재요청 필요 | 요청 카드의 현재 상태와 사용 가능한 버튼 | **접수** 상태의 **요청 취소**를 사용한 뒤 다시 요청 |
| 요청 반려 | 요청 카드의 상태와 메모 | 요청 번호와 메모를 확인하고 필요하면 지원 요청 |
| 관리자 메뉴가 없음 | 현재 계정과 역할 | Super Admin에게 권한 확인 |
| 요청 상태가 멈춤 | 요청 번호와 마지막 변경 시각 | [지원 요청](support.md) |

브라우저를 새로 고치기 전에 입력한 내용을 별도로 안전하게 메모하세요. 인증번호나 가족 사진은 지원 요청에 첨부하지 마세요.
